> Run CommonJS script in sandbox with customized style

## Installation

```bash
npm install my-runner
# or use yarn
yarn add my-runner
```

## Usage

```javascript
const myRunner = require('my-runner')

myRunner.runFile('/path/to/file.js').module.exports
```
